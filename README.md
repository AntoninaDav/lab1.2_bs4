## Выводы по работе

Подводя итог, можно сказать, были получены теоретические знания и применены практические навыки. 
В результате парсинга и анализа данных были получены сведения о рецептах блюд из курицы, включая время приготовления и количество ингредиентов. Построение графика ТОП-5 самых быстрых рецептов позволило наглядно определить блюда, которые требуют минимальных временных затрат, что может быть полезно для быстрого выбора рецепта. Визуализация данных упростила сравнение рецептов и повысила наглядность результатов парсинга.
